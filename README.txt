
Ferramenta para copiar arquivos de uma pasta origem para um pasta destino,
renomeando os arquivos(.mp3/.m4a) para uma sequência numérica, partindo de 1.

A finalidade da renomeação dos arquivos é copiar arquivos de música para
dispositivos móveis antigos que não dão suporte à longos nomes, como os
geralmente encontrados em arquivos de música.
